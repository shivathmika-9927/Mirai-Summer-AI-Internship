# 🌌 AI Multiverse

An interactive **AI-powered chatbot application** built with **Streamlit** and **Google Gemini AI**.  
AI Multiverse allows users to explore multiple AI personalities across different universes such as **Student**, **Programming**, **Emotional Support**, **Career**, and **Entertainment**. Users can also create their own custom AI personality with personalized behavior and communication style.

## ⚙️ Installation

### Clone the repository

```bash
git clone https://github.com/shivathmika-9927/Mirai-Summer-AI-Internship.git
```

### Navigate to the project

```bash
cd Assignment2_AI_Multiverse
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Create a `.env` file

```env
GEMINI_API_KEY=YOUR_GEMINI_API_KEY
```

### Run the application

```bash
streamlit run app2.py
```
